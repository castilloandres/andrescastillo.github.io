How to count symbols in a string in JavaScript

10/12/2018

By: Andres Castillo Ormaechea

The first thing we tend to think when we need to count the number of symbols in a string is the following:

> 'A'.length
1
> '1234'.length
4
> 'symbols'.length
7

Nevertheless using .length fails on the following examples:

> 'ð€'.length
2
> 'ðŸ’©'.length
2
> 'ð¯ •'.length
2

This is because using .length counts unicode code points and not symbols. If we want to count symbols as far as readable text goes for most international character sets more or less the approach would be something like:

> Array.from('ð€').length
1
> Array.from('ðŸ’©').length
1
> Array.from('ð¯ •').length
1

Let's go over the terms first

  • Code unit: its the unit of storage of code points. In UTF-8 this means 8-bits, in UTF-16 is 16-bits.
  • Code point: is a number that represents a symbol or part of a symbol by the Unicode standard. A text/string is a sequence of unicode code points.
  • Symbol: is a single graphical unit that can represent a grapheme or part of a grapheme.
  • Grapheme: is a single graphical unit that the reader recognizes visually as a single element of the written system.

So to summarize it in another way:

  • Strings can be composed of multiple graphemes.
  • Graphemes can be composed of multiple symbols.
  • Symbols can be composed of multiple code points.
  • Code points can be composed of multiple code units/bytes.
