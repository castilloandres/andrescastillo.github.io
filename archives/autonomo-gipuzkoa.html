Autónomo en Gipuzkoa

  • Existen dos impuestos principales IVA e IRPF.
  • IVA: impuesto de comercio.
  • IRPF: impuesto sobre la renta de personas físicas. Impuesto sobre tu utilidad anual declarado trimestralmente.
  • Primeros tres años el IRPF es 7%.

Tramos anuales de IRPF para el 2018.

  • 19% de IRPF => para bases imponibles entre 0€ y 12450€
  • 24% de IRPF => para bases imponibles entre 12450€ y 20200€
  • 30% de IRPF => para bases imponibles entre 20200€ y 35200€
  • 37% de IRPF => para bases imponibles entre 35200€ y 60000€
  • 45% de IRPF => para bases imponibles de 60.000€ o superiores
