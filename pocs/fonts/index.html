Page Font Size

List of use cases:

  • Bad font size (< 21px).
  • Good font size (>= 21 px).
  • Same font and background color (invisible text is considered).
  • Adding a viewport tag fixes everything.
  • `opacity:0` text.
  • `visibility:hidden` text.
  • `display:none` text.
  • Adding an h1 and h2 tag makes the font size test pass (seems like some sort of heuristics are applied).
